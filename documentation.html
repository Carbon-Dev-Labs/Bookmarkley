Getting Started

Lets get started with Bookmarkley. Getting started with Bookmarkley is fairly simple, if you follow the instructions down below you will be all set.

Bookmarkley is your key to configuration and customization of any webpage. Before we Get Started, heres what Bookmarkley does, and does not do.

  • Bookmarkley Is/Does
  • Temporarily modify the webpage
  • A bookmarklet menu/tool
  • A Bookmarked Script
  • Free
  • Bookmarkley Isn't/Doesn't
  • Edit the webpage for anyone else
  • A Paid Software
Bookmarkley is still in beta and high development so it might not work on some webpages/the style might be weird.
Bookmarkley is a temporary script and will go away when you refresh the page (Unless you use the Keep Background options.)

Instalation

All you have to do to install Bookmarkley is just add the bookmark and your all set! Bookmarkley is just a bookmark with a script so when you use it, it injects the script into the webpage and modifys the page content. This is temporary and will go away when you refresh it. Show Me How

If Bookmarkley does not load imediatly, be patient. I host the script on a server that doesn't run 24/7 so its likley that the server is starting.

Setting up and Personalizing Bookmarkley

Lets personalize Bookmarkley to suit you. Personalizing Bookmarkley looks a lot different than other software/extensions. To personalize Bookmarkley, we are gonna use a tool we made to generate a bookmark for you. Since Bookmarkley isn't installed locally, we store your settings in the script so every time you use it, it loads your configuration when loading the script. Show Me How

Once you have customized and setup your custom Bookmarkley configuration, you are probably wondering how you put your configuration into play. After you generate your custom script, copy that script. If you already have a Bookmarkley bookmark, edit that bookmark and replace the existing script with your new one. If you don't already have a Bookmarkley bookmark, either follow the earlier instructions or create a new bookmark yourself, name it whatever, and then copy your custom script into the URL input.

How Does it Work

Overview

Bookmarkley is a bookmarklet script. When you click the bookmark, it injects a custom script into any webpage you are on and modifys the content.


Telemetry


What data do we collect?

Bookmarkley does not track you. The only information we collect is,

  • How many times you use Bookmarkley
  • What time you are using Bookmarkley
  • Little IP Address details (Just General location nothing much)
Nothing we collect is connected to any of our users or stored under accounts. We don't store anything linked to any of our users, nor do we sell or distrubte any of this information.


Why do we collect this stuff?

We collect a tiny bit of information from each user so we can build graphs and data to see what our users are like. We use this information to make Bookmarkley all the better.


I want out. What can I do?

Don't want to be apart of our Telemetry? Simple, all you have to do is use the Customize Bookmarkley tool and disable Telemetry.
